What's the logic behind the Bazaar?

Having pre-designed a multitude of templates and add-ons, you can simply choose your desired website components and let me assemble the final product for a fraction of the price of a built-from-scratch website.

In the end, you will receive a folder (repository) containing all the code files that the website you purchased consists of. Moreover, the code is well documented so, if you want to edit some of it (make sure you make a back-up first), you can identify what is written where.

However, if you do not find any of the templates appealing, by following the right path on the Home page you can order a website from scratch.

From this point on, you can do whatever you like with it. You can either give the repository to a back-end engineer to host it, or you can host it yourself by using sites such as Netlify.

To conclude, as I always say, someone's work speaks louder than their words. So, if you still need proof of quality, the Front-End Bazaar website's written entirely by me, as are the templates and add-ons.
